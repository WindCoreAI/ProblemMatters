## Usage Instructions

### Running Research

1. Copy the command from the "Command" column
2. Run it in Claude Code: paste the command and press Enter
3. Wait for the research to complete
4. Update the checklist with results (see below)

### Updating After Research

After completing research for a field, update the corresponding row:

1. Change status from `‚¨ú` to `‚úÖ`
2. Add the date in "Last Updated" column (YYYY-MM-DD format)
3. Add the number of problems found in "Problems Found" column
4. Update the "Quick Stats" table at the top

**Example update:**
```markdown
| ‚úÖ | Web Development | `/discover-problems technology-software software-engineering web-development` | 2026-01-20 | 8 |
```

### Command Options

You can add options to customize research:

- `--max N` - Set maximum problems to discover (default: 10)
- `--depth LEVEL` - Set depth: shallow, standard, deep, exhaustive

**Examples:**
```bash
/discover-problems technology-software ai-ml generative-ai --max 15
/discover-problems healthcare clinical-care --depth deep
/discover-problems financial-services fintech --max 20 --depth exhaustive
```

### Domain-Level Research

To research an entire domain (all fields), omit the field:
```bash
/discover-problems technology-software software-engineering
/discover-problems healthcare clinical-care
```

### Industry-Level Research

To research an entire industry (all domains), use only the industry:
```bash
/discover-problems technology-software
/discover-problems healthcare
```
